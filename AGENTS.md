## Project overview

This project is a Waku framework project that is a quote generator that can be exported to a image.

## How does it work?

The user customize the name, headline, photo, logo, colors, font and the text/quote. Think that the resulting image is destined to social media.

## Stack

- Waku framework
- The global state is managed by Zustand
- Styling with pure CSS and using modern CSS like layers, reset, CSS nesting, light-dark function
- Take accesibility and responsive into 100% priority
